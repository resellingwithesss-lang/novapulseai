# NovaPulseAI Client

Frontend app for NovaPulseAI (Next.js App Router).

## Local Run

From repository root:

- `npm run dev:client`

Or from `client/`:

- `npm run dev`

Default URL:

- `http://localhost:3000`

## Quality Checks

- `npm run lint`
- `npm run typecheck`

## Environment

Copy:

- `client/.env.example` -> `client/.env.local`

Key vars:

- `NEXT_PUBLIC_API_URL`
- `NEXT_PUBLIC_APP_URL`
- `NEXT_PUBLIC_GOOGLE_CLIENT_ID`

For full stack setup, see the root README at `../README.md`.
